POSIX标准

最近在提交exfatprogs代码的时候，作者指出没有正确的使用返回值，并给我举了示例：aeb200d#diff-0b6fa8c11476e438a96f6a7b6e2f77df6857a10fface89a66b974d2dbda872ffR1063-R1125

所以我查了一下最新的POSIX标准：POSIX.1-2024

POSIX.1-2024 正式发布！POSIX.1-2024 是 IEEE Std 1003.1 的最新版本，由 IEEE 和 The Open Group 联合制定。该标准是 POSIX® （Portable Operating System Interface of Unix，可移植操作系统接口）标准的一部分，定义了操作系统与应用程序之间的接口标准，旨在提高软件的可移植性。

官网如下：https://pubs.opengroup.org/onlinepubs/9799919799/

exit code

根据文档：为了让wait或者waitpid能够成功捕获返回值，exit code的返回值应该是0，EXIT_SUCCESS，EXIT_FALURE或者其他的值（范围在0~255之间）【文档里面0377指的是八进制】。

exfatprogs maintainer建议：0 on success, 1 on generic error, 2 on usage error
