1.3 开始运行qemu

关键参数解释：

  • -kernel: 指定编译好的内核二进制文件。
  • -initrd: 指定我们刚做的内存文件系统。
  • -nographic: 不启动图形界面，直接将输出重定向到当前的终端。
  • -append "console=ttyS0": 告诉内核把启动日志输出到串口（即你的终端）。

💡 小提示：如果你想退出 QEMU 窗口，按下 Ctrl + A 然后按 X 即可。

2.qemu中如何支持exfat文件系统并默认挂载一个镜像

2.1 开启exfat支持

🛠️ 第一步：打开内核菜单配置

回到你的 linux 内核源码目录，运行图形化配置菜单：

🛠️ 第二步：找到并开启 exFAT 选项

在弹出的蓝色菜单界面中，使用键盘的上下方向键移动，回车键进入子菜单，按两次 Esc 返回上一级。请按照以下路径寻找：

  1. 向下滚动，进入 File systems --->
  2. 向下滚动，进入 DOS/FAT/EXFAT/NT Filesystems --->
  3. 找到 exFAT filesystem support 这一项。

关键操作：

  • 停在这一项上，按键盘上的空格键，直到它前面的括号变成 **[\*]**。
    • [*] 代表直接编译进内核（推荐）。
    • <M> 代表编译成独立的模块（exfat.ko）。
  • 配置好后，使用左右方向键在底部选择 < Save >，按回车保存默认的文件名（.config），然后一直选择 < Exit > 退出菜单。

🛠️ 第三步：重新编译内核

因为我们只修改了少量的配置，这次编译会非常快，内核只会增量编译涉及到的文件：
