2.node page相关理解

2.1 node page是什么

node block对应的是磁盘上对应的结构，而真实存在于内存中，准确的说应该是缓存Page Cache中的应该是页。

f2fs_get_node_page函数会将4K的node block转换成4K的node page。

2.2 node page对应的address_space
